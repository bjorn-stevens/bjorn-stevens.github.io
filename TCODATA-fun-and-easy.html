Fun and Easy TCO data¶

This short slide deck illustrates how well structured data combined with an intake catalog can be easy to browse and explore. Topics can be explored in more depth by clicking down, or a new topic can be opened by clicking right.

Opening the catalog¶

The Tropical Cloud Observations Group of the Max Planck Institute for Meteorology decided to serve all of its data using zarr over html. The data is compiled (or is being compiled) as a single data set, using intake and the catalog:

"http://tcodata.mpimet.mpg.de/catalog.yaml"

Presently only a few instruments have been added, more or on their way.

Below we give some examples showing how to navigate through the catalog. Next we give some examples using the data files that we identified by browsing.

In [1]:
import intake
cat = intake.open_catalog("http://tcodata.mpimet.mpg.de/catalog.yaml")

Inspecting the radar data¶

First we look at a time-slice of the radar date, randomly picking a 2 day period in early february. Then we zoom in on an interesting echo.

In [5]:
import matplotlib.pyplot as plt # for making plots
import seaborn as sns # for making plots prettier

radar = cat.BCO.radar_MBRS_c1_v1.to_dask()
radar.sel(time=slice("2023-03-02","2023-03-03"),alt=slice(0,17000)).Ze.plot(x="time",cmap="turbo",figsize=(15,5))
sns.despine(offset = 10)

Combining instruments¶

While the radar makes it look like a long period of rain, most of the reflectivities near the surface would hardly be perceptible as such. Below we plot the near-surface reflectivities as a time-series along with some measurements with the weather station.

In [11]:
radar = cat.BCO.radar_MBRS_c1_v1.to_dask().sel(time=slice("2023-03-03T01","2023-03-03T03"))
wxt   = cat.BCO.surfacemet_wxt_v1.to_dask().sel(time=slice("2023-03-03T00","2023-03-03T03"))

fig, axs = plt.subplots(3,1, sharex=True, figsize = (10,3),constrained_layout=True)
axs[0].plot(wxt.time,wxt.T)
axs[1].plot(wxt.time,wxt.RI)
axs[2].plot(radar.time,radar.Ze[:,2])

axs[0].set_ylabel('T / degC') 
axs[1].set_ylabel('$P$ / mm hr$^{-1}$') 
axs[2].set_ylabel('$Z_\mathrm{e}$ ') 
axs[2].set_xlabel('UTC time (2023-03-03)')
